## 📖 Sobre o Projeto

O **AgroNexo** é uma plataforma web desenvolvida para auxiliar produtores rurais na gestão eficiente de suas propriedades.

O sistema centraliza informações do rebanho, controle financeiro e indicadores produtivos em uma interface moderna, intuitiva e responsiva, permitindo uma tomada de decisão mais rápida e baseada em dados.

## ⚙️ Configuração

### 1. Clone o repositório

```bash
git clone https://github.com/seuusuario/agronexo.git
```

### 2. Acesse a pasta

```bash
cd agronexo
```

### 3. Configure o Supabase

Substitua as credenciais pelas do seu projeto:

```javascript
const SUPABASE_URL = 'https://seuprojeto.supabase.co';
const SUPABASE_ANON_KEY = 'sua-chave-anon';
```

### 4. Execute

Abra o arquivo:

```bash
index.html
```

ou utilize uma extensão como **Live Server**.
